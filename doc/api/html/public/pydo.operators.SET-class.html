Class SET

object --+
         |
        SET

a way of passing a set into PyDO where clauses (for IN), e.g.: 
>>> IN(FIELD('foo'), SET('spam', 'eggs', 'nougat'))
(foo IN ('spam', 'eggs', 'nougat'))
